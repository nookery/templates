超大屏幕

欢迎登陆页面！

这是一个超大屏幕（Jumbotron）的实例。

学习更多

为了获得占用全部宽度且不带圆角的超大屏幕，请在所有的 .container class 外使用 .jumbotron class

      <div class="jumbotron">
         <div class="container">
            <h1>欢迎登陆页面！</h1>
            <p>这是一个超大屏幕（Jumbotron）的实例。</p>
            <p><a class="btn btn-primary btn-lg" role="button">
               学习更多</a>
            </p>
         </div>
      </div>

徽章（Badges）

与标签相似，主要的区别在于徽章的边角更加圆滑。

主要用于突出显示新的或未读的项。如需使用徽章，只需要把 <span class="badge"> 添加到链接、Bootstrap 导航等这些元素上即可。
